showpath
========

Show various linux'ish system paths

Usage
=====
# show default set of paths
% showpath

# show python paths
% showpath-python

# show rvm related paths
% showpath-rvm

# show gem related paths
% showpath-gem

# show java related paths
% showpath-java


Install
=======

Copy "bin/\*" to your path.

If you don't know what is on the path, run 'showpath'
